# Responsive Recipe Website

A responsive recipe website developed using **HTML, CSS, and JavaScript**, consuming data from the **TheMealDB API**.

This project follows a **mobile-first approach**, with a layout built from scratch and all recipe data dynamically rendered using JavaScript.

---

## Introduction

This project was developed as part of **Module 2 of the ElevateHer program**.

The main goal was to build a recipe website focused on **JavaScript, API integration, and responsive design**, applying real-world web development practices.  
All recipe information — such as image, name, ingredients, instructions, and video — is **fetched from an external API** and displayed dynamically on the page.

The interface was created from scratch using **semantic HTML** and **custom CSS**, with a clean layout optimized for mobile devices.
